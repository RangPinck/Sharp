ctrl + f - поиск в указанной области
ctrl + h - замена в указанной области
ctrl + c - копирование выделенного отрезка или строки на которой стоит указатель
ctrl + v - вставка последнего скопированного элемента из буфера обмена
win + v - вставка выбранного элемента из буфера обмена (его истории)
ctrl + x - вырезание выделенного отрезка или строки на которой стоит указатель
ctrl + shift + end - выделение всего, что находится ниже курсора
ctrl + shift + home - выделение всего, что находится выше курсора
ctrl + a - выделение всего
tab - табуляция или раскрытие выбранного модуля (кто знает, тот поймёт)
shift + f3 - отключение капслока на выделенном тексте
shift + alt + вверх/низ - копирование в верх или вниз строки на которой стоит корсор 
shift + alt + вправо/влево - расширение выделенной области
alt + стрелочки - перемещение строки вверх или вниз
ctrl + вверх/вниз - прокрутка страницы без перемещения курсора
alt + право/лево - перемещение между 2 вкладками, которые посещались недавно
выделение + tab - добавление табуляции выделенным строкам
shift + tab - обрантная операция верхней
ctrl + d - выделение слова на котором курсор
ctrl + shift + l - выделение указанного слова во всём коде + доп курсоры к этим словам   
ctrl + g - перехождение к конкретной строке кода
ctrl + b - открытие и закрытие боковой панели
ctrl + shift + e - быстрое открытие панели проводника (explorer)
ctrl + ё/~/` - открытие и скрытие терминала
ctrl + f4 / ctrl + w - закрыие и открытие файла
alt + цифра - перемещение между открытыми файлами
ctrl + pgdown/pgup - перемещение между двумя соседними файлами
так же можно перемещаться - ctrl + tab
ctrl + p - открытие файла из папки проекта
ctrl + m tab - перемещение по меню
